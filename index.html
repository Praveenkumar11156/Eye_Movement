step 1 : Create an HTML page with two images, one representing the eye and another representing the pupil.

step 2 : Use CSS to position the pupil image inside the eye image, so that the pupil appears to be moving inside the eye.

step 3 : In JavaScript, use an event listener to detect mouse movement on the page.

step 4 : When the mouse moves, calculate the position of the mouse relative to the eye image.

step 5 : Use this position to calculate the new position of the pupil image inside the eye image.

step 6 : Update the CSS position of the pupil image to reflect the new position.
